# Cakeshop-Chatbot
I have  developed a RAG-based AI chatbot for a potential client i.e. for a cake shop and have integrated it into a customized website. The chatbot provides instant, accurate answers to customer queries about cakes prices, flavors, availability, shop timings, and answers others queries  by retrieving verified information from a vector database. The chatbot is fully integrated into the website through an embedded chat widget, offering a seamless experience. This solution enhances customer service, reduces repetitive inquiries, provides 24/7 support, and can be easily adapted for caf√©s, bakeries, restaurants, and small retail businesses.
